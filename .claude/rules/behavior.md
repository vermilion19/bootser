# 행동 규칙

## 커밋 규칙

- 커밋은 반드시 사용자 허락 후 진행
- "커밋할까요?" 질문 → 승인 받은 후에만 커밋
- 사용자가 명시적으로 "커밋해", "commit" 등 지시한 경우에만 진행

## AOP 체크리스트

코드 작성 후 AOP 어노테이션이 있으면 호출 경로 검토:

- [ ] **Self-invocation 체크**: 같은 클래스 내에서 `@Cacheable`, `@Transactional`, `@Async` 등 AOP 어노테이션이 붙은 메서드를 호출하는지 확인
  - 같은 클래스 내 호출 → 프록시 안 거침 → AOP 무시됨
  - 해결: 다른 Bean에서 호출하도록 분리
